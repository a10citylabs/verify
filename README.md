# C2PA Content Credentials Verifier

Verify content credentials in every format the C2PA reader supports:

- **Images** — JPEG, PNG, WebP, AVIF, HEIC/HEIF, GIF, SVG, TIFF, camera RAW (DNG, NEF, ARW)
- **Video** — MP4, MOV (QuickTime), AVI
- **Audio** — MP3, WAV, M4A
- **Documents** — PDF, XML/XHTML
- **C2PA manifest stores** — `.c2pa` files
